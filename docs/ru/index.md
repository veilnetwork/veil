# Документация Veil (OVL1)

Veil — децентрализованная гибридная сеть с открытым исходным кодом на Rust, реализующая протокол OVL1.

## Версия протокола

Текущая версия OVL1: **v1** (magic `0x4F564C31`, version byte `0x01`).

IPC-протокол: версия **1** (`IPC_PROTOCOL_VERSION = 1`).
